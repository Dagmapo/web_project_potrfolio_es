## Mi Portafolio Personal

### Descripción del Proyecto

Portafolio personal diseñado para showcasear mis proyectos, habilidades y experiencia como desarrollador Full-Stack Junior. El sitio presenta una interfaz moderna y responsiva que destaca los proyectos realizados, las tecnologías dominadas y proporciona formas de contacto directo a través de múltiples canales.

### Tecnologías y Técnicas Utilizadas

- **HTML5** - Estructura semántica y accesibilidad
- **CSS3** - Diseño responsivo con Flexbox y Grid, variables CSS, media queries
- **Fuentes personalizadas** - Archivo Black y Open Sans para tipografía profesional
- **Diseño Responsive** - Soporte para desktop, tablet y mobile
- **Git y GitHub** - Control de versiones y publicación con GitHub Pages
- **Figma** - Diseño y prototipado
